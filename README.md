# PROBLEM 1:
 - Given an string find the first character that has recurred and print it if none of the characters recurred then print message "No recurring character".
 
	Example:

		string = "ABCA"
		output: A
		
		string = "BAACB"
		output: B
		
		string = "ABC"
		output: No recurring character
